Projet IN104 en C : transcription de notes de piano

# Utilisation

## Installation dépendences

| Librairies |
| ---------- |
| GSL        |
| SDL2       |

* Linux : 
```
$ sudo apt install libgsl-dev
$ sudo apt install libsdl2-dev
```

## Compilation et exécution

```
$ chmod +x ./compilation.sh
$ ./compilation.sh
```
